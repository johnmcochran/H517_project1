About the Project

In 1854, a cholera epidemic led to the innovation of a visualization that assisted in finding the source of the outbreak. Dr. John Snow created a visualization which enabled the discovery of the source, a single water pump. This discovery led to improvements in public health policies related to water sanitation. This project seeks to recreate the visualization using d3.js.

The data was provided by Professor Reda in the following format:

        Map coordinates in the form of a json file with a list of arrays of points representing a segment, where each data point is a dictionary with an x value and a y value.

[

[

//segment 1 WHAT THE HELL

{x1,y1},

{x2,y2},

{x3,y3}

],

 

[

// segment 2

{x1,y1},

{x2,y2}

],

...

]

        Pump locations in csv file with x and y coordinates.

        Date with corresponding number of deaths in csv file

        Information about individual deaths were provided in csv file with x,y coordinates, age, and gender.
